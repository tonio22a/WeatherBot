# Weather Bot 

Telegram-бот для получения актуальной информации о погоде по геолокации пользователя. Бот предоставляет подробный прогноз с использованием данных OpenWeatherMap.

## 📋 Описание проекта

Проект представляет собой простого Telegram-бота, который помогает пользователям быстро узнать погоду в их текущем местоположении. Основные возможности:

- Получение погоды по геолокации пользователя
- Детальная информация о температуре, влажности, скорости ветра
- Визуальное отображение погоды с помощью эмодзи
- Простой и понятный интерфейс с кнопками
- Информация о проекте

При нажатии на кнопку "Моя геопозиция" бот запрашивает доступ к местоположению и отправляет пользователю актуальные данные о погоде

## 🛠️ Технологии

Проект создан с использованием следующих технологий и библиотек:

- **Python 3.11+** - основной язык программирования
- **pyTelegramBotAPI** - библиотека для работы с Telegram Bot API
- **OpenWeatherMap API** - внешний сервис для получения данных о погоде

## 🚀 Запуск проекта

### Предварительные требования

- Установленный Python 3.11 или выше

### Пошаговая инструкция

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/tonio22a/WeatherBot.git
   cd weather-bot
   ```

2. **Установите необходимые зависимости**
   ```bash
   pip install pyTelegramBotAPI requests
   ```

3. **Получите токен для бота**
   - Найдите в Telegram бота [@BotFather](https://t.me/botfather)
   - Отправьте команду `/newbot` и следуйте инструкциям
   - Скопируйте полученный токен

4. **Создайте файл конфигурации**
   - Создайте файл `config.py` в корневой папке проекта
   - Добавьте в него следующий код:
   
   ```python
   TOKEN = "ваш токен"
   ```

5. **Запустите бота**
   ```bash
   python bot.py
   ```
   В консоли должно появиться сообщение "бот запущен"

6. **Найдите своего бота в Telegram** и отправьте команду `/start`

### Альтернативный способ (без файла config.py)

## 🎯 Использование

1. Запустите бота командой `/start`
2. Нажмите кнопку **"Моя геопозиция"**
3. Нажмите на скрепку и отправьте свою геолокацию
4. Получите подробную информацию о погоде:
   - Название города
   - Описание погоды с эмодзи
   - Температура и ощущаемая температура
   - Влажность воздуха
   - Скорость ветра

Также доступна кнопка **"О проекте"** для получения информации о назначении бота.
